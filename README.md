# 跨境经贸企业与产品资料管理系统（Flask）

本系统定位为**企业资料与产品资料的轻量化后台管理平台**，用于服务跨境经贸贸促会场景下的信息沉淀与长期留存，核心目标是：

- 企业信息录入、查询、维护
- 产品信息录入、查询、维护
- 附件资料归档与备份

> 系统已做简化重构，前台仅保留三个核心模块：**企业库、产品库、备份管理**。

## 1. 核心模块

### 企业库
- 管理企业基础档案、联系人、主营方向。
- 集成展示企业层面的资质、外贸能力、合作需求与风险备注等信息（不再作为独立主导航模块）。

### 产品库
- 管理产品主档、规格参数、贸易条款、认证与包装物流信息。
- 保留原有字段能力，页面按简化后台样式展示。

### 备份管理
- 管理员可手动备份数据库与上传文件目录。
- 启动时仍支持“每日一次”数据库自动备份逻辑。

## 2. 导航与页面结构

- 左侧固定 Sidebar：`首页 / 企业库 / 产品库 / 备份管理`
- 顶部仅保留系统名与当前页面标题
- 首页 Dashboard 仅展示：
  - 企业数量
  - 产品数量
  - 已上传文件数量
  - 最近新增企业
  - 最近新增产品

## 3. 兼容性说明（重要）

为避免破坏历史数据，数据库模型未进行激进删除：

- 外资客户、外资需求、撮合匹配、项目进展、出海分析等历史表与字段仍保留。
- 本次重构以“**隐藏复杂模块入口 + 简化主流程**”为主，最大限度确保旧数据可持续读取。

## 4. 运行方式

### 4.1 安装依赖

```bash
pip install -r requirements.txt
```

### 4.2 启动

```bash
python app.py
```

启动后访问：

- 首页：`http://127.0.0.1:5000/`
- 登录：`http://127.0.0.1:5000/登录`

## 5. 默认账号

- 管理员：`admin / admin123`
- 普通用户：`user / user123`

> 生产环境请立即修改默认密码。

## 6. 备份与恢复

### 手动备份（管理员）
访问 `/backup`（或 `/backups`）可执行：
- 数据库备份：`backups/backup_db_YYYYMMDD_HHMMSS.sqlite`
- 上传目录备份：`backups/backup_uploads_YYYYMMDD_HHMMSS.zip`

### 恢复数据库
1. 停止服务
2. 备份当前库：`cp trade_agent.db trade_agent.db.bak`
3. 覆盖恢复：`cp backups/backup_db_xxx.sqlite trade_agent.db`
4. 重启：`python app.py`
